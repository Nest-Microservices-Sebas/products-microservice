# Product Microservice

## Dev
1. Clonar el repo.
2. Instalar dependencias.
3. Crear un archivo `.env` con las variables de entorno.
4. Ejecutar migracion de prisma `npx prisma migrate dev`
5. Ejecutar `npm run start:dev`
